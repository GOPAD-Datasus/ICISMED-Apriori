# ICISMED Apriori

## Paper

> This repository is based on our paper:  
**A Hybrid Approach Based on Clustering and Association Rules to Analyze Sociodemographi Profiles Related to Newborn Health and Well-being**   
Authors: Morsoleto, R. et al.  
Presented and accepted at: [ICISMED](https://www.icismed.org/) 2025.

## Installation & Usage

Poetry was used for dependency management. To download it, visit: [python-poetry.org](https://python-poetry.org/)

Activate a virtual environment and execute:

```bash
poetry install
# Or
pip install .
```


```bash
python main.py
```
